環境構築手順

トップ | 環境構築手順 | CI/CD 流れ

手順 1: Docker Desktop のインストール

公式サイトから OS に合わせてインストール

手順 2: リポジトリをクローン


git clone <リポジトリURL>
cd live-test_server

手順 3: test 環境を起動


cd test
docker compose up -d

手順 4: live 環境を起動


cd ../live
docker compose up -d

※ test 環境で動作確認後に live 環境へデプロイするのが基本フローです。
